# Análisis del efecto del tratamiento A sobre la presión arterial

**Autor:** Jesus Rodrigo Huaripaucar Chalco

## Objetivo

Evaluar si el tratamiento A reduce la presión arterial sistólica (PAS) en comparación con el tratamiento B, ajustando por edad, sexo, IMC, colesterol, tabaquismo y diabetes.
